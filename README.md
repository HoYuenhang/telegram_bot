# 一个自用 telegram bot

有需要可拿去用，项目中使用的外部api token需自己去相关网站申请

# 食用方法：

1. 安装好nodeJs

2. npm 安装相关插件，相关插件请查看package.json文件

```"dependencies": {
"dependencies": {
        "axios": "^0.26.0",
        "md5": "^2.3.0",
        "node-telegram-bot-api": "^0.56.0"
    },
"devDependencies": {
        "@types/axios": "^0.14.0",
        "@types/node-telegram-bot-api": "^0.56.0"
    }
```

3. 使用命令 node app.js 启动项目
